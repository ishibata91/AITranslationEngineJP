# Detail Specs

この directory は、各ページの機能要件と非機能要件の正本を置く。
active exec-plan で固定した要件を close 前に昇格させ、後続 task の参照起点にする。

## Naming

- 詳細仕様は `docs/detail-specs/<page-name>.md` を正本とする

## Pages

- [`master-dictionary.md`](./master-dictionary.md)

## Notes

- plan close 前に、承認済みの機能要件と非機能要件だけを移す
- 実装手順や一時判断は active / completed exec-plan に残す
