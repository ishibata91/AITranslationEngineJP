# Wireframes

この directory は、`docs/spec.md` と `docs/er.md` の現行仕様から起こした low-fidelity ASCII wireframe を置く。
画面遷移や情報配置の叩き台だけを扱い、visual design や細部文言は確定しない。

## Screen List

- `app-shell.md`: 単一ウィンドウの共通 shell
- `foundation-data.md`: マスターペルソナとマスター辞書の観測
- `input-review.md`: import 済み入力データの確認
- `job-setup.md`: 翻訳ジョブ作成と AI 基盤選択
- `job-run.md`: 翻訳ジョブの進捗観測と制御
- `output-review.md`: 翻訳結果と出力成果物の観測
