## 記法ルール

- ケース ID は `SCN-<topic-abbrev>-NNN` 形式で固定する
- `NNN` は `001` から始める 3 桁連番とする
- `ケース一覧` には少なくとも `正常系`、`主要例外系`、`状態遷移`、`責務境界` を 1 件以上含める
- Markdown table は使わず、1 ケースごとの縦型ブロックで記載する
- `期待結果` は観測可能な結果として書き、説明 prose だけで曖昧にしない
