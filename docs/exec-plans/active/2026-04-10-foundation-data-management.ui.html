Visible Outcomes

  • collection summary と entry count を先頭で観測する
  • 中央一覧と右詳細は selected collection に同期して入れ替える
  • 未選択時は detail placeholder を出し、編集導線は disabled とする
  • 編集導線と Rebuild 導線は詳細 pane 下部にまとめる
