# Active Plans

関連文書: [`../../index.md`](../../index.md), [`../../core-beliefs.md`](../../core-beliefs.md)

このディレクトリには、未完了の計画を置く。

## Rules

- 非自明な変更は、実装前にここへ短い計画を追加する
- 実装や設計内包タスクは `templates/impl-plan.md`、修正タスクは `templates/fix-plan.md` を使う
- ファイル名は `YYYY-MM-DD-topic.md` を基本とする
- 1 つの計画には、目的、対象、受け入れ条件、更新対象文書を含める
- 実装 task で必要な `UI` / `Scenario` / `Logic` は plan 本文の section として保持する
- 完了したら `../completed/` へ移動し、結果を残す
