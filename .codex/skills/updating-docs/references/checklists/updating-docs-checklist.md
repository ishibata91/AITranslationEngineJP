# Updating Docs Checklist

## Knowledge Check

- [ ] Copilot completion report を確認した
- [ ] human approval record を確認した
- [ ] approved artifact と canonical target を対応づけた
- [ ] validation 結果と remaining gaps を記録した

## Common Pitfalls

- [ ] Copilot 修正完了前に正本化しなかった
- [ ] 未承認 draft を正本化しなかった
- [ ] implementation-scope を自動昇格しなかった
- [ ] workflow 変更や product 実装を混ぜなかった
