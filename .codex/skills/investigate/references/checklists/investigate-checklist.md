# Investigate Checklist

## Knowledge Check

- [ ] 観測事実、UI evidence、仮説、未観測 gap を分けた
- [ ] evidence path、再現条件、UI check scope を残した
- [ ] design continuation に必要な risk を返した

## Common Pitfalls

- [ ] evidence なしの結論を書かなかった
- [ ] UI check 専用 agent 前提にしなかった
- [ ] implementation-time investigation を扱わなかった
