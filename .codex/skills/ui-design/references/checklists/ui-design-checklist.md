# UI Design Checklist

## Knowledge Check

- [ ] Figma file URL、node id、review frame を確認した
- [ ] state、variant、responsive、overflow risk を確認した
- [ ] repo-side artifact は参照、判断、証跡に限定した

## Common Pitfalls

- [ ] Figma 以外へ UI 構造正本を分散しなかった
- [ ] product frontend implementation に踏み込まなかった
- [ ] human review が必要な visual decision を隠さなかった
