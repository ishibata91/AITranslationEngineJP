# UI Design Checklist

## Knowledge Check

- [ ] HTML mock path、preview URL、review screenshot を確認した
- [ ] state、variant、responsive、overflow risk を確認した
- [ ] `ui-design.md` は参照、判断、証跡に限定した

## Common Pitfalls

- [ ] HTML mock 以外へ UI 構造正本を分散しなかった
- [ ] product frontend implementation に踏み込まなかった
- [ ] human review が必要な visual decision を隠さなかった
