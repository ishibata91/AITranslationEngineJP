# Propose Plans Checklist

## Knowledge Check

- [ ] memory、`.codex/README.md`、`docs/index.md` を確認した
- [ ] active / completed task folder に同種 task がないか確認した
- [ ] distiller、designer、investigator の必要判定を最初に明示した
- [ ] spawn packet が context 継承なしで足りる内容になっている
- [ ] human review と人間向け Copilot handoff を分けた

## Common Pitfalls

- [ ] spawned agent に会話文脈を暗黙継承させなかった
- [ ] diagram を designer の資料作成 scope から外さなかった
- [ ] Codex から Copilot へ直接 handoff しなかった
- [ ] Copilot 修正完了前に正本化へ進まなかった
