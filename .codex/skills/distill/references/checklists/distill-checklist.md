# Distill Checklist

## Knowledge Check

- [ ] 圧縮粒度が `catalog`、`summary`、`full` のどれかとして説明できる
- [ ] 重要な fact に根拠 path がある
- [ ] `confirmed`、`inferred`、`gap` が混ざっていない

## Common Pitfalls

- [ ] focused skill が持つ詳細観点を共通 skill に戻していない
- [ ] implementation-scope 承認後の実装前整理を扱っていない
- [ ] product code、product test、docs 正本の変更に進んでいない
